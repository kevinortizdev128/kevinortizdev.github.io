Proyecto Mascotas

Descripción: Sistema para el registro y consulta de información de mascotas, incluyendo datos de dueños, especie, raza y vacunación.
Tecnologías: HTML, CSS, JavaScript.
Características: Validación de formularios, búsqueda rápida de mascotas y un diseño intuitivo para facilitar el uso.
