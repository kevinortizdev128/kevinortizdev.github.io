Proyecto Librería

Descripción: Aplicación web para la gestión de libros y autores, permitiendo registrar obras, autores, realizar búsquedas y administrar el inventario de manera eficiente.
Tecnologías: HTML, CSS, JavaScript.
Características: Alta, baja y modificación de libros; buscador dinámico; y una interfaz amigable para el usuario.
